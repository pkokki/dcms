Aging buckets

Aging buckets are time periods used to review and report on open receivables.

Scoring parts

A scoring part uses a rule, a virtual resource or an expression to return a score value.

Scorers

(a) Scorers determine transaction status: current, delinquent, and pre-delinquent.

(b) Scorers determine the value of each customer at an operational data level.

Letter templates

A letter template provides the form and text for the correspondence.

Uses a virtual resource to get the customer-related data that appears in the letter.

Letter plans

A plan is composed of one or more score ranges and correspondence templates associated with aging bucket lines.

Job tasks

Typically, a job task is a long-running, data-intensive task.
